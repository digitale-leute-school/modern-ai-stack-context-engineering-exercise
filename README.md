# Modern AI Stack: Context Engineering Exercise

A small Next.js starter app for training people to use coding agents with better project context.

The app is a simple **Team Directory**. Participants will ask a coding agent to add search and department filtering twice:

1. once with only the feature request
2. once after creating project context, such as `CLAUDE.md`

## Getting started

```bash
npm install
npm run dev
```

Open <http://localhost:3000>.

## Useful commands

```bash
npm run dev        # start local dev server
npm run build      # build production app
npm run lint       # run linting
npm run typecheck  # run TypeScript checks
```

## Important note

This repository intentionally does **not** include a `CLAUDE.md`. Participants should create one during the exercise.

A suggested reference is available in [`CLAUDE.example.md`](./CLAUDE.example.md).
